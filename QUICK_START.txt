
## ✨ All Your Commands

### Learning
- `"learn that [X] is [Y]"` → Teach fact
- `"actually [X] is [Y]"` → Correct fact
- `"what is [X]"` → Ask about learned facts

### Visualization
- `"visualize"` or `"show workspace"` → ASCII grid
- `"plot"` or `"graph"` → Matplotlib chart

### Personality
- `"tell me a joke"` or `"be funny"` → Robot humor
- `"tell me something"` → Fun facts
- `"hello"` / `"hi"` → Mood-based greetings

### ARM (Existing)
- `"move left/right/up/down/forward/back"`
- `"grab [object]"` / `"release"`
- `"add [name] at [distance]cm [angle]degrees"`
- `"camera on"` / `"camera off"`
- `"list"` / `"scan"` / `"find [object]"`

### Data (Existing)
- `"save"` / `"remember"`
- `"voice on"` / `"voice off"`
- `"help"` / `"status"`
